# Global-Superstore-Sales-Analysis-using-Power-BI
This project focuses on analyzing the Global Superstore dataset using Power BI to generate meaningful insights about sales, profit, and customer behavior. The dashboard helps in understanding business performance across different regions, categories, and time periods.

# 📊 Global Superstore Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard built using the Global Superstore dataset. The goal is to analyze sales performance, customer behavior, and business insights across different regions, categories, and time periods.

## 📁 Dataset Details
•  Dataset: Global Superstore
•  Includes:
  ○  Sales, Profit, Quantity
  ○  Customer Name & Segment
  ○  Product Category & Sub-category
  ○  Order Date & Ship Date
  ○  Region, State, City
  ○  Shipping Mode

## 🛠️ Tools & Technologies
Power BI
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)

## 📊 Dashboard Features
•  Interactive slicers (Region, Category, Year)
•  KPI Cards (Total Sales, Total Profit, Total Orders)
•  Sales trend analysis (Line Chart)
•  Category & Sub-category analysis (Bar Chart)
•  Profit distribution (Pie/Donut Chart)
•  Geographic visualization (Map)
•  Top 5 Customers analysis

## 📌 Key Insights
•  Identified top-performing regions and products
•  Found most profitable categories
•  Analyzed customer contribution to revenue
•  Observed yearly sales growth trends
•  Compared shipping modes and delivery efficiency
