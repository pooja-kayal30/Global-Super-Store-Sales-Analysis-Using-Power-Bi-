# 📊 Global Superstore Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive Power BI dashboard built using the Global Superstore dataset. The goal is to analyze sales performance, customer behavior, and business insights across different regions, categories, and time periods.

## 📁 Dataset Details
The dataset includes the following key fields:
- Order ID, Order Date, Ship Date
- Customer Name, Segment
- Country, City, Region, Market
- Product Category & Sub-Category
- Sales, Profit, Quantity, Discount, Shipping Cost

## 📊 Dashboard Features
- 💰 Total Sales, Profit, Quantity KPI Cards
- 🌍 Region-wise and Country-wise Sales Analysis
- 📦 Category & Sub-Category Performance
- 📈 Sales Trend over Time (Year/Month)
- 🚚 Shipping Cost & Order Priority Analysis
- 👥 Customer Segment Insights

## 🛠️ Tools & Technologies
- Power BI (Data Visualization)
- DAX (Data Analysis Expressions)
- Excel (Data Source)

## 📌 Key Insights
- Identified top-performing regions and markets
- Analyzed profit vs discount impact
- Found high-sales product categories
- Tracked sales growth trends over time
